# 🔐 Secure Login & Authentication System

## 📌 Project Overview
A secure user authentication system built from scratch as a **C++ Console Application**. 
Instead of focusing on a graphical user interface (GUI), this project dives deep into **core programming logic, data validation, and security principles**.

The primary goal of this project was to implement robust manual testing and quality assurance practices at the foundational level of the code.

## ⚙️ Key Features
- **User Authentication:** Secure login mechanism validating user credentials.
- **Input Validation:** Prevents common input errors and unauthorized access attempts.
- **Edge-Case Handling:** Extensively tested with invalid inputs and unexpected user behaviors to ensure system stability.
- **Terminal-Based Execution:** Focuses purely on backend logic and core functionality without UI distractions.

## 🧪 Testing & Quality Assurance
A significant portion of this project's development was dedicated to software testing. I acted as my own QA tester by intentionally attempting to "break" the system. 
- Conducted rigorous manual testing in the command line.
- Tested out-of-bound inputs, special characters, and empty fields.
- Ensured the program logic does not crash under unexpected conditions.

## 🛠️ Technologies Used
- **Language:** C++
- **Environment:** Command Line Interface (CLI)
